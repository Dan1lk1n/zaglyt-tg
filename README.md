# zaglyt-tg

Генеративный Telegram-бот, который обучается на истории сообщений чата и на их основе создаёт свои, стараясь сохранять контекст сообщения, на которое он отвечает.

---

## Как это работает (Текущая версия)

1. **Контекстный поиск (BM25):** Когда пользователь пишет сообщение, бот анализирует его и ищет в базе данных чата наиболее подходящие по смыслу реплики с помощью алгоритма ранжирования BM25.
2. **Стемминг (Snowball):** Для поиска используются корни слов (стеммы), что позволяет находить совпадения независимо от окончаний.
3. **Генерация (Цепь Маркова):** На основе найденных BM25 сообщений строится локальная цепь Маркова, которая генерирует новый связный ответ.

## Запуск проекта

1. Убедитесь, что у вас установлен Go (версии 1.18 или новее).
2. Склонируйте репозиторий.
   ```bash
   git clone https://github.com/pip64/zaglyt-tg
   ```
3. Создайте таблицу `channels` в вашей PostgreSQL базе данных:
   ```
   CREATE TABLE IF NOT EXISTS channels (
        id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
        channel_id BIGINT NOT NULL UNIQUE,
        enabled BOOLEAN NOT NULL DEFAULT TRUE,
        mode VARCHAR(50) DEFAULT NULL
   );
   CREATE INDEX IF NOT EXISTS idx_channels_channel_id ON channels(channel_id);
   ```
4. Настройте конфигурацию в файле .env на примере .env-example
5. Запустите проект:
   ```bash
   go run .
   ```
